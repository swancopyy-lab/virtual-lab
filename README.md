# Virtual Lab Simulator

This repository contains a Python Flask virtual laboratory simulator.

## Features
- Interactive simulations for various laboratory experiments.
- User-friendly interface for students and educators.
- Easy to set up and run locally.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/swancopyy-lab/virtual-lab.git
   ```
2. Navigate to the project directory:
   ```bash
   cd virtual-lab
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
- Run the application:
  ```bash
  python app.py
  ```
- Access the simulator in your browser: `http://127.0.0.1:5000/`
